# Virtual environment

1. Create virtual environment : python -m venv myvenv

2. Activate virtual environment: myvenv\Scripts\activate

## What is the function for this?

- a best practice that helps you manage dependencies and avoid conflicts across projects.
- keeps your development environment clean, organized, and easier to maintain

## How to run the python project
1. Activate the virtual environment: myvenv\Scripts\activate
2. Run the python file: python filename.py
